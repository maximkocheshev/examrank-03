# âœ… Exam_Rank-03
The aim of the projects is to write a program that will read an "operation file" and print the result in the terminal.
## Micro_paint
Example operation file:
```
10 10 _
r 1.2 1.2 3.01 3.02 j
R 5.2 5.2 7.01 7.02 j
```
## Compilation
```
gcc micro_paint.c
```
## Usage
```
./a.out <operation file>
```
## Output
```
__________
__________
__jjj_____
__j_j_____
__jjj_____
__________
______jjjj
______jjjj
______jjjj
______jjjj
```
